# 🧮 Calculator App

A modern and responsive calculator built using HTML, CSS, and JavaScript. 
This project provides a clean user interface for performing basic arithmetic operations with both button and keyboard support.

---

## 🚀 Features

* 🧮 Basic Arithmetic Operations
* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 📊 Percentage Calculation
* 🗑️ Delete Last Character
* 🔄 Clear All (AC)
* ⌨️ Keyboard Support
* 🎨 Modern Dark Theme UI
* 📱 Fully Responsive Design

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Grid Layout + Flexbox)
* JavaScript (DOM Manipulation)

---

## 🎨 Features Overview

### 🧮 Calculator Display

* Large display screen
* Real-time input and output
* Error handling for invalid expressions

### ⌨️ Keyboard Support

Supports:
* Numbers (0-9)
* Arithmetic Operators (+, -, *, /, %)
* Decimal Point (.)
* Enter Key (=)
* Backspace (Delete)
* Escape (Clear)

### 🎨 User Interface

* Modern dark theme
* Smooth hover animations
* Responsive button grid layout

---

## 📱 Responsive Design

* ✅ Desktop
* ✅ Tablet (768px)
* ✅ Mobile (480px)

## 💡 How to Run

1. Clone the repository:

```
git clone https://github.com/Sivaponnuvel/Calculator-Project
```

2. Open the project folder

3. Run `index.html` in your browser
